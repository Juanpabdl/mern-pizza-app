# Mozzirella's Pizza App

This project is a web application for a fictional bussiness called Mozzirella's Pizza, that allows for 
users to make orders out of the restaurant´s menu, pay for them and track their orders in real time.

Currently on development.

## Key features

1. The website can be accessed by a user account, classified into standard and admin accounts
  - Standard users can create their own accounts on the site with basic functionality
  - Administrators get higher access to the website´s functionality, their accounts are set up after approval

2. Users can explore the restaurant´s menu, selecting dishes, combos and types of dishes in further detail

3. Users can create an order by selecting dishes / combos from the menu and adding them to the shoppig cart.

4. Once a dish has been selected, the users can confirm the order and select a payment method.

5. After payment, the user creates an order, and can track its status in current time, alongside an estimated time for completion.

6. The Admin has a list of pending orders, and can select one to showcase its description

7. The Admin can change the status of the selected order as it progresses, sharing the status to the corresponding customer. Once an order is marked as Completed, the order will no longer appear on the order list.

8. The Admin gets access to the restaurant´s menu and can modify the dishes and combos prices and descriptions.
